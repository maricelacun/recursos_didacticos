¿Qué son las estructuras repetitivas?

Las estructuras repetitivas, también conocidas como bucles o loops, son fundamentales en programación. Permiten ejecutar un bloque de código múltiples veces mientras se cumpla una condición determinada.

Escucha esta introducción a los bucles en Java

En Java existen tres tipos principales de estructuras repetitivas:

  • while - Ejecuta un bloque mientras se cumpla una condición
  • do-while - Similar al while, pero garantiza al menos una ejecución
  • for - Ideal cuando sabemos exactamente cuántas veces se debe repetir

¿Por qué son importantes los bucles?

Imagina que necesitas imprimir los números del 1 al 100. Sin bucles, tendrías que escribir 100 líneas de código. Con bucles, puedes hacerlo en solo 3 líneas:

for(int i = 1; i <= 100; i++) { System.out.println(i); }

Estructura while

El bucle while ejecuta un bloque de código mientras una condición sea verdadera.

while (condición) { // Código a ejecutar }

Características del while:

  • Primero evalúa la condición, luego ejecuta el código
  • Si la condición es falsa desde el principio, el código no se ejecuta
  • Debes asegurarte de que en algún momento la condición sea falsa para evitar bucles infinitos

Diagrama de flujo de la estructura while

Ejemplo práctico

int contador = 1; while (contador <= 5) { System.out.println("Iteración número: " + contador); contador++; // Incrementamos el contador }

Estructura do-while

El bucle do-while es similar al while, pero con una diferencia clave: el código se ejecuta al menos una vez antes de verificar la condición.

do { // Código a ejecutar } while (condición);

Características del do-while:

  • El bloque de código se ejecuta primero, luego se evalúa la condición
  • Garantiza al menos una ejecución del bloque de código
  • Útil cuando necesitas ejecutar el código al menos una vez, como en menús interactivos
int numero; do { System.out.print("Ingresa un número entre 1 y 10: "); numero = scanner.nextInt(); } while (numero < 1 || numero > 10); System.out.println("Correcto! Elegiste: " + numero);
Presiona "Simular entrada" para ver cómo funciona

El flujo del do-while siempre ejecuta el bloque al menos una vez antes de verificar la condición.

¿Cuál es la principal diferencia entre while y do-while?

El while evalúa la condición después de ejecutar el bloque
El do-while siempre ejecuta el bloque al menos una vez
El do-while no puede tener condiciones complejas
No hay diferencia, son exactamente iguales

Estructura for

El bucle for es ideal cuando sabemos exactamente cuántas veces queremos que se ejecute un bloque de código.

for (inicialización; condición; actualización) { // Código a ejecutar }

Partes del bucle for:

  • Inicialización: Se ejecuta una vez al inicio del bucle
  • Condición: Se evalúa antes de cada iteración
  • Actualización: Se ejecuta al final de cada iteración

Ejemplo básico

for (int i = 0; i < 5; i++) { System.out.println("Iteración: " + i); }

For mejorado (for-each)

Java ofrece una versión simplificada para recorrer arrays y colecciones:

String[] frutas = {"Manzana", "Banana", "Cereza"}; for (String fruta : frutas) { System.out.println(fruta); }
