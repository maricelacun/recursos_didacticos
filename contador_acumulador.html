Acumuladores y Contadores en Java

Domina el uso de estas herramientas esenciales para trabajar con bucles

¿Qué son acumuladores y contadores?

Cuando trabajamos con bucles en Java (como for, while o do-while), a menudo necesitamos llevar un registro de ciertos valores durante las iteraciones. Aquí es donde entran en juego los acumuladores y contadores.

Imagina que estás en un supermercado:

1
El contador sería como el número de productos que llevas en tu carrito
2
El acumulador sería como el total de dinero que vas gastando

Contadores

Un contador es una variable que incrementa o decrementa su valor en una cantidad constante (generalmente 1) con cada iteración del bucle.

Características principales:

  • Se usa para contar ocurrencias o iteraciones
  • Normalmente se incrementa en 1 (pero puede ser otro valor)
  • Se inicializa generalmente en 0 o 1 según el caso

Ejemplo básico:

int contador = 0; // Inicialización
for (int i = 0; i < 10; i++) {
    contador++; // Incremento
}
System.out.println(contador); // Muestra 10

Ejemplo práctico:

// Contar números pares del 1 al 100
int pares = 0;
for (int num = 1; num <= 100; num++) {
    if (num % 2 == 0) {
        pares++;
    }
}
System.out.println(pares); // Muestra 50

Acumuladores

Un acumulador es una variable que va sumando (o restando, multiplicando, etc.) valores variables durante las iteraciones del bucle.

Características principales:

  • Se usa para acumular valores (sumas, productos, etc.)
  • El valor que se suma puede variar en cada iteración
  • Se inicializa generalmente en 0 (para sumas) o 1 (para productos)

Ejemplo básico:

int suma = 0; // Inicialización
for (int i = 1; i <= 5; i++) {
    suma += i; // Acumulación
}
System.out.println(suma); // Muestra 15 (1+2+3+4+5)

Ejemplo práctico:

// Calcular el promedio de notas
double sumaNotas = 0.0;
int[] notas = {85, 90, 78, 92, 88};
for (int nota : notas) {
    sumaNotas += nota;
}
double promedio = sumaNotas / notas.length;
System.out.println(promedio); // Muestra el promedio

Comparación entre contadores y acumuladores

Característica Contador Acumulador
Propósito Contar ocurrencias Acumular valores
Incremento Valor constante (generalmente 1) Valor variable
Inicialización típica 0 o 1 0 (sumas), 1 (productos)
Operación típica ++ o -- +=, -=, *=, /=
Ejemplo de uso Contar estudiantes aprobados Sumar edades para calcular promedio

Errores comunes y cómo evitarlos

¡Cuidado con estos errores!

!
Olvidar inicializar la variable (puede contener basura)
!
Inicializar con un valor incorrecto (ej. suma=1 para acumular)
!
Poner la inicialización dentro del bucle (se reinicia en cada iteración)
!
Confundir cuando usar cada uno (contar vs acumular)

Ejemplo de error común:

// ERROR: suma se reinicia en cada iteración
for (int i = 1; i <= 5; i++) {
    int suma = 0; // MAL: inicialización dentro del bucle
    suma += i;
}
// suma no estará accesible aquí y se pierde en cada iteración
