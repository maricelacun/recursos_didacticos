Suma de Pares

Escribe un programa en Java que solicite al usuario un número entero N y calcule la suma de todos los números pares desde 1 hasta N.

// Ejemplo de entrada/salida
Si N = 10, la salida debe ser: 30 (2+4+6+8+10)
Si N = 7, la salida debe ser: 12 (2+4+6)
2

Verificador de Palíndromos

Crea un programa en Java que pida al usuario ingresar una cadena de texto y determine si es un palíndromo (se lee igual de izquierda a derecha que de derecha a izquierda), ignorando mayúsculas, minúsculas y espacios.

// Ejemplo de entrada/salida
Entrada: "Anita lava la tina", Salida: Es palíndromo
Entrada: "Java", Salida: No es palíndromo
3

Calculadora de Factorial

Desarrolla una función en Java que calcule el factorial de un número entero no negativo. El factorial de N (N!) es el producto de todos los enteros positivos menores o iguales a N. (0! = 1)

// Ejemplo de entrada/salida
Si N = 5, N! debe ser: 120 (5*4*3*2*1)
Si N = 0, N! debe ser: 1
4

public class Contacto { ... }

Conversor de Temperatura

Escribe un programa en Java que permita convertir temperaturas entre Celsius y Fahrenheit. El usuario debe poder elegir la dirección de la conversión e ingresar el valor.

// Fórmulas:
