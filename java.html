¿Qué es Java?

Java es un lenguaje de programación orientado a objetos, creado en 1995 por la empresa Sun Microsystems, y ahora es mantenido por Oracle.

El lema de Java

"Escribe una vez, ejecútalo en cualquier lugar". Esto significa que, si tú escribes un programa en Java, puede funcionar en cualquier sistema operativo: Windows, Mac, Linux... sin cambiar nada.

¿Dónde se usa Java?

Aplicaciones móviles

Android está basado en Java. Millones de apps usan Java como lenguaje principal.

Software de escritorio

Desde editores de texto hasta complejos sistemas de diseño asistido por computadora.

Aplicaciones web

Sitios dinámicos con tecnologías como Java EE, Spring y muchos frameworks más.

¿Cómo funciona Java?

Java no se ejecuta directamente en el sistema operativo como otros lenguajes. Sigue un proceso especial:

1

Archivo .java

Tú escribes tu programa en un archivo con extensión .java que contiene el código fuente.

2

Compilación

El compilador javac traduce tu código a un lenguaje intermedio llamado bytecode.

3

Archivo .class

El bytecode se guarda en un archivo .class, que no es entendible por humanos pero es portable entre sistemas.

4

Ejecución

La JVM (Java Virtual Machine) interpreta el bytecode y lo ejecuta en cualquier sistema operativo.

Ventaja clave

Gracias al bytecode y la JVM, los programas Java pueden funcionar en Windows, Linux, Mac o incluso en teléfonos Android, sin cambiar una sola línea de código.

¿Qué puedes hacer con Java?

Java es tan versátil que lo puedes usar en casi cualquier área de desarrollo:

  • Aplicaciones móviles: Android está basado en Java.
  • Aplicaciones de escritorio: como editores de texto, calculadoras, etc.
  • Aplicaciones web: sitios dinámicos con Java EE o Spring
  • Videojuegos: muchos juegos 2D y 3D utilizan Java o librerías como LibGDX.
  • Programación de hardware: por ejemplo, con Raspberry Pi
  • Sistemas empresariales complejos: como los bancos y aseguradoras.

Comunidad y recursos

Java tiene una gran comunidad global, muchos recursos para aprender y una enorme cantidad de librerías y frameworks que facilitan el trabajo.

Tu primer programa en Java

Vamos a escribir el clásico "Hola Mundo" para que veas lo sencillo que puede ser:

public class HolaMundo {
  public static void main(String[] args) {
    System.out.println("¡Hola, mundo Java!");
  }
}

Para ejecutarlo:

  1. Guarda el código en un archivo llamado HolaMundo.java
  2. Compílalo con: javac HolaMundo.java
  3. Ejecútalo con: java HolaMundo
